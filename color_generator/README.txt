	Last Modified: December 11, 2019

	Gets textarea input on button click or
	enter key press. If blank, generates a
	random hexadecimal color value.

	Red, Green, and Blue values are stored.
	Gets two triadic (adjacent R, G, or B
	int) values, then displays a table.

	New triadic color value is converted
	from RGB format back to hexadecimal,
	then formatted for display.

	Main color value and both triadic
	values are used to generate three dark-
	to-light color palettes, nine shades
	each.

	All rgb values are converted to
	rgb(255, 255, 255) format for CSS styling
	(main button, triadic display table),
	and (255, 255, 255) format for text
	display.
